## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/jagwazorld-max/JagX-monie.git
   ```
2. Navigate to the project directory:
   ```bash
   cd JagX-monie
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
